Ingeniero de Software Senior

Tech Solutions Inc. | Remoto

Descripción del puesto

Tech Solutions Inc. está buscando un Ingeniero de Software Senior altamente motivado y experimentado para unirse a nuestro equipo remoto. El candidato ideal tendrá una sólida formación en desarrollo de software, con experiencia en el diseño, desarrollo e implementación de soluciones escalables y de alto rendimiento. Serás responsable de liderar proyectos, mentorizar a ingenieros junior y colaborar con equipos multifuncionales para entregar productos de alta calidad.

Responsabilidades

  • Diseñar, desarrollar y mantener aplicaciones web con tecnologías modernas.
  • Colaborar con producto y diseño para definir y entregar nuevas características.
  • Escribir código limpio, eficiente y bien documentado.
  • Realizar code reviews y dar feedback constructivo al equipo.

Requisitos

  • Licenciatura en Informática o campo relacionado.
  • 5+ años de experiencia en desarrollo de software.
  • Experiencia con React/Angular/Vue u otros frameworks de JavaScript.
  • Conocimiento de metodologías ágiles y control de versiones (Git).

Acerca de la empresa

Tech Solutions Inc. crea soluciones de software de vanguardia para distintas industrias. Fomentamos un entorno colaborativo e inclusivo con oportunidades de crecimiento profesional, salarios competitivos y beneficios integrales.
